# Royal Events Catering Website

Welcome to the **Royal Events Catering Website** — a modern, responsive, and professionally designed platform built to showcase catering and event management services. This project represents a complete front-end solution for a growing events business, combining clean design, structured content, and interactive user experience.

---

## Project Overview

Royal Events is a catering and event planning brand focused on delivering high-quality services for both corporate and social occasions. This website serves as the digital presence of the organization, allowing potential clients to explore services, learn about the company, and easily get in touch.

The platform is designed with simplicity, accessibility, and professionalism in mind, ensuring a seamless browsing experience across all devices.

---

## Project Objectives

- Create a professional online presence for Royal Events  
- Showcase services in a clear and visually appealing way  
- Provide an easy-to-use contact system for clients  
- Deliver a responsive and modern user interface  
- Build a scalable structure for future upgrades  

---

## Key Features

### Navigation System
- Fully functional navigation bar linking all pages  
- Sticky and responsive design for better usability  
- Clean layout aligned with modern web standards  

### Home Page
- Hero section with strong branding and messaging  
- Call-to-action (CTA) encouraging user engagement  
- Overview of services and company highlights  

### About Page
- Detailed company background  
- Mission, vision, and values  
- Professional layout reflecting brand identity  

### Services Page
- Grid-based service cards with equal height alignment  
- Services include:
  - Corporate Catering  
  - Social Events  
  - Staff Meals  
  - Event Management  
  - Event Decoration  
  - Utensils Hire  
- Hover effects for improved interactivity  
- Responsive design for all screen sizes  

### Blog Page
- Blog-style layout with featured posts  
- Sidebar for categories and recent posts  
- Structured content for future expansion into a full blog system  

### Contact Page
- Functional contact form with validation  
- JavaScript-powered form submission feedback  
- Auto-refresh functionality after submission  
- Separate refresh button for user control  
- Contact details clearly displayed  
- Logo section instead of map for branding focus  

### Call-To-Action (CTA) Section
- Positioned above the footer  
- Encourages users to request services  
- Displays key contact information  

### Footer Section
- Multi-column layout including:
  - Company overview  
  - Quick navigation links  
  - Services list with anchor links  
  - Contact information  
- Fully responsive and edge-to-edge design  
- Consistent styling across all pages
